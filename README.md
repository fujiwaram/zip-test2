# zip-test2
golang test code that create zip file by using GCS streaming.

## Overview

<img src="./doc/overview.drawio.svg"/>

## Usage

### Start GCS Emulator
```
make up
```

### Run memory saving
```
make run
```

### Run waste memory
```
make runw
````

### Stop GCS Emulator
```
make down
```
